what is java & who is using it?

java can be broadly defined as,a general purpose ,object -oriented,platform independent,concurrent programming language that runs vey fast.

general purpose implies, that it is not constrained to one particular domain and can be used to develop wide variety of applications.

java is an oop language.object orientedness helps model the real world scenarios in a more natural way.

javas mantra is write once,run anywhere(wora) ie you write a program once and must be able to run it on any platform regardless of the os architecture.and this is referred to as platform independence.

java supports concurrent programming, which is nothing but multi-threading, and this enables programs to perform multiple activities at around the same time.java is also very fast.it is almost indistinguishable from c and c++.

These are few core attributes of java this makes java , a really solid and a very reliable programming language.

also java is free thus individual and startups can launch their own products without investment.

initially java was developed to run inside embeded systems and later within web browsers in the form of java programs called applets.but it soon evolved as a popular language for developing large scale web applications, stand alone desktop applications and mobile applications.
