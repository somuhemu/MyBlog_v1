what is React?

React is a javascript library/framework created and maintained by facebook that is used for building user interfaces.So its technicallya UI library.

The diff bw react and angular is that react at its core is only responsible for creating user interfaces, it doesn't have things like router or an http client built in, but we have popular packages corresponding to it.

React gives us a way to build websites & UIs with organized and reusable Components.

The components can include the output,which is basically the html in react to use sth called jsx, the logic or the javascript, and can also include the css within your components.so the htmlI(jsx),css,js all are integrated ok.

so with react and other frontend frameworks we look at our UI as different components like the searchbar,header,footer,blogpost so eth we look at as a component.
